# stencil-react-vue-ionic-example

**Setup:**

`yarn install`

`cd packages/web-components`

`yarn build`

**After you build the Stencil.js web components:**

For Vue:

`cd packages/vue-site/`

`yarn serve`

For React:

`cd packages/react-site/`

`yarn start`

For Ionic:

`cd packages/ionic-site/`

`yarn start`

For Storybook:

`cd packages/storybook/`

`yarn storybook`
